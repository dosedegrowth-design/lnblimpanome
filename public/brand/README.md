# Brand Assets — LNB

## Como atualizar o logo oficial

O arquivo `lnb-logo.svg` atual é um **placeholder**. Substitua pelo arquivo oficial:

```bash
# Coloque o arquivo recebido nesta pasta:
cp ~/Downloads/lnb-logo-oficial.png public/brand/lnb-logo.png
# ou
cp ~/Downloads/lnb-logo-oficial.svg public/brand/lnb-logo.svg
```

## Cores oficiais da marca

- **Azul LNB** (primária): `#2EB1E2`
- **Cinza chumbo** (secundária): `#3A3A3A`
- **Branco**: `#FFFFFF`
- **Off-white** (background suave): `#F8FAFC`

Definidas em `src/app/globals.css` como CSS variables Tailwind v4.
